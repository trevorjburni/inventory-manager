# Inventory Manager for hair salon

This app is designed for a specific hair salon, to help them keep track of products they use for their customers hair appointments.

# Technology Used

- JavaScript
- Node
- React
- MongoDB
- HTML
- Bulma
- CSS
- Heroku
- Passport Authentication (incomplete)
and more..
